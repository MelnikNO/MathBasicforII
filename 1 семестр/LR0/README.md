### Задание 1.2: Операции с векторами

#### Сложение векторов

```python
sum_ab = a + b                # [4, 6]
```


Сложение векторов выполняется поэлементно. Результат геометрически соответствует диагонали параллелограмма, построенного на векторах `a` и `b`.

#### Умножение на скаляр

```python
scaled_a = 2 * a              # [6, 8]
```



Умножение вектора на скаляр изменяет его длину без изменения направления (при положительном скаляре).

---

### Задание 1.3: Скалярное произведение и норма

**Скалярное произведение:**

```python
dot_product = np.dot(a, b)    # 11
```

Скалярное произведение показывает степень сонаправленности векторов.

**Норма (длина) вектора:**

```python
norm_a = np.linalg.norm(a)    # 5.000
norm_b = np.linalg.norm(b)    # 2.236
```

**Угол между векторами:**

```python
cos_angle = dot_product / (norm_a * norm_b)  # 0.984
angle_deg = np.degrees(np.arccos(cos_angle)) # 10.3°
```

**Вывод:** Угол между векторами `a` и `b` составляет **10.3°**, что подтверждает их высокую степень сонаправленности.

## Часть 2: Работа с матрицами

### Задание 2.1: Создание матриц

**Созданные матрицы:**

```python

```

| Матрица | Размерность |
|---------|-------------|
| A | `(2, 2)` |
| B | `(2, 2)` |
| C | `(2, 3)` |

---

### Задание 2.2: Операции с матрицами

#### Сложение матриц

```python
sum_AB = A + B
```

Результат:

```
[[ 6  8]
 [10 12]]
```

#### Умножение матрицы на скаляр

```python
scaled_A = 3 * A
```

Результат:

```
[[ 3  6]
 [ 9 12]]
```

#### Умножение матриц

```python
product_AB = A @ B
```

Результат:

```
[[19 22]
 [43 50]]
```

#### Проверка некоммутативности

```python
product_BA = B @ A
```

Результат:

```
[[23 34]
 [31 46]]
```

**Вывод:** `A*B != B*A` — умножение матриц **не коммутативно**.

---

### Задание 2.3: Умножение матрицы на вектор

```python
result_Aa = A @ a  # [11, 25]
```



**Геометрический смысл:** Матрица `A` действует как линейное преобразование, переводящее вектор `a` в новый вектор `[11, 25]`. Это демонстрирует, как матрицы могут выполнять повороты, масштабирование и другие преобразования пространства.

### Задание 3.3: Простая линейная регрессия с матрицами

Была реализована модель линейной регрессии для предсказания цены дома по площади, используя метод наименьших квадратов.

**Данные:**

| Площадь (кв.м) | Цена (млн руб.) |
|----------------|-----------------|
| 50 | 3.2 |
| 60 | 3.8 |
| 70 | 4.1 |
| 80 | 4.7 |
| 90 | 5.2 |
| 100 | 5.8 |
| 110 | 6.1 |
| 120 | 6.9 |

**Решение:**

```python
w = (X^T * X)^(-1) * X^T * y
```

**Результаты:**

| Параметр | Значение |
|----------|----------|
| Свободный член (intercept) | `0.624` |
| Коэффициент при площади | `0.051` |
| Уравнение модели | `Цена = 0.624 + 0.051 × Площадь` |
| Предсказание для дома 95 кв.м | `5.49 млн руб.` |

### Задание 1: Евклидово расстояние

```python
def euclidean_distance(vec1, vec2):
    """Вычисляет евклидово расстояние между двумя векторами"""
    return np.linalg.norm(vec1 - vec2)
```

**Результат:** расстояние между `[3, 4]` и `[1, 2]` = `2.828`

**Применение:** Евклидово расстояние — это длина прямой между двумя точками в пространстве. Широко используется в задачах кластеризации (k-means) и классификации (k-NN).

---

### Задание 2: Нормализация вектора

```python
def normalize_vector(vec):
    """Нормализует вектор (приводит к единичной длине)"""
    norm = np.linalg.norm(vec)
    if norm == 0:
        return vec
    return vec / norm
```

**Результат:** нормализованный вектор `[3, 4]` = `[0.6, 0.8]`

**Применение:** Нормализация приводит вектор к единичной длине, сохраняя его направление. Используется в задачах, где важна только ориентация вектора (например, косинусное сходство).

### Задание 3: Матрица поворота

**2D матрица поворота:**

```python
def rotation_matrix_2d(angle_deg):
    """Создает матрицу поворота на заданный угол в градусах"""
    angle_rad = np.radians(angle_deg)
    return np.array([
        [np.cos(angle_rad), -np.sin(angle_rad)],
        [np.sin(angle_rad), np.cos(angle_rad)]
    ])
```

**Результат:** поворот вектора `[3, 4]` на 45° даёт `[-0.707, 4.95]`

### Задание 5: Исследование различных метрик расстояний

Были исследованы следующие метрики расстояний:

| Метрика | Формула | Значение |
|---------|---------|----------|
| Евклидово (L2) | √Σ(xᵢ - yᵢ)² | 5.196 |
| Манхэттенское (L1) | Σ|xᵢ - yᵢ| | 9.000 |
| Косинусное | 1 - cos(θ) | 0.025 |
| Минковского (p=3) | (Σ|xᵢ - yᵢ|³)^(1/3) | 4.327 |

**Применение в Data Science:**

| Метрика | Применение |
|---------|------------|
| **Евклидово (L2)** | Кластеризация (k-means), метод k-ближайших соседей, регрессия |
| **Манхэттенское (L1)** | Устойчиво к выбросам, LASSO-регрессия |
| **Косинусное** | Анализ текстов, поиск похожих документов, рекомендательные системы |
| **Минковского** | Обобщение L1 и L2 с параметром p для различных задач |
