## Архитектура MLP

### Реализация с нуля

MLP реализован вручную на Python с использованием только NumPy. Основные компоненты:

#### 1. Инициализация (Xavier/Glorot)

```python
limit = np.sqrt(6.0 / (layer_sizes[i] + layer_sizes[i+1]))
w = np.random.uniform(-limit, limit, (layer_sizes[i], layer_sizes[i+1]))
b = np.zeros((1, layer_sizes[i+1]))
```

#### 2. Функции активации

| Слой | Функция | Формула |
|------|---------|---------|
| Скрытые | ReLU | $f(x) = \max(0, x)$ |
| Выходной | Sigmoid | $\sigma(x) = \frac{1}{1+e^{-x}}$ |

#### 3. Функция потерь (Binary Cross-Entropy + L2)

```python
loss = -np.mean(y_true * np.log(y_pred) + (1 - y_true) * np.log(1 - y_pred))
l2_reg = (self.l2_lambda / 2) * sum(np.sum(w ** 2) for w in self.weights)
```

#### 4. Backpropagation

Реализован полный цикл обратного распространения ошибки с градиентным спуском.

## Подбор оптимального порога

### Методология

Для бинарной классификации с сигмоидным выходом порог по умолчанию равен 0.5. Однако в медицинских задачах часто важно минимизировать пропуски (False Negative), поэтому был проведен подбор оптимального порога на валидационной выборке.

**Критерий:** Максимизация F1-score

### Результаты подбора

| Порог (τ) | F1-score | Precision | Recall |
|-----------|----------|-----------|--------|
| 0.50 (стандартный) | 0.827 | 0.902 | 0.764 |
| **0.456 (оптимальный)** | **0.876** | **0.827** | **0.931** |

**Улучшение:** F1-score увеличился на **4.87%**

### Критерий Юдена

Оптимальный порог соответствует критерию Юдена:

$$J = TPR - FPR = 0.6389$$
